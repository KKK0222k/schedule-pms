# Schedule Management Project

## 1. Overview
This is a schedule management project built with:
- Java 1.8
- Maven
- Spring / eGovFramework
- MyBatis mapping
- Oracle Database
- Frontend: HTML/JSP, Vanilla JS, jQuery, FullCalendar

## 2. Directory Structure Guide
- `/src/main/java`: Backend controllers, services, common components
- `/src/main/resources/egovframework/spring`: Spring DB/Transaction context
- `/src/main/resources/egovframework/sqlmap`: MyBatis mappers
- `/src/main/webapp/WEB-INF/config/app`: App configs
- `/src/main/webapp/js/schedule/*`: Frontend logic modules based on requested path structure
- `/docs/schema.sql`: Database schema definition
